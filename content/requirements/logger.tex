\subsection{Logger}

\subitem{
    The Logger records mission data and provides mechanisms for transmission or storage by collecting telemetry, writing to storage, and/or sending data via communications links. The Logger is comprised of two submodules: Storage and Transmitt. Each submodule handles data persistence and transmission for mission telemetry and diagnostic information.
}

\subsubsection{Interfaces}
\paragraph{Inputs}
\begin{enumerate}
    \item Telemetry streams and configuration parameters
\end{enumerate}
\paragraph{Outputs}
\begin{enumerate}
    \item Stored logs and transmitted telemetry packets
\end{enumerate}
\paragraph{Internal}
\begin{enumerate}
    \item Internal logging control signals
\end{enumerate}

\subsubsection{Storage}
\subitem{
    The Storage submodule is responsible for persisting telemetry and logs to onboard non-volatile memory,
    ensuring data is retained for later retrieval and analysis.
}

\begin{enumerate}

    \item Consume telemetry streams to be persisted, storage configuration as defined in XXX:location
    \item Output stored log files or records retrievable by time/index as defined in XXX:location

    \item Execute within allocated CPU and power budgets.

\end{enumerate}

\subsubsection{Transmitt}
\subitem{
    The Transmitt submodule handles transmission of selected telemetry data to ground stations or
    other external systems via available communications links.
}

    \item Consume selected telemetry streams and transmission policy as defined in XXX:location
